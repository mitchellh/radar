- (Object) data_extension(klass)

Adds a data extension to an application. Data extensions allow extra data to be included into an Radar::ExceptionEvent (they appear in the Radar::ExceptionEvent#to_hash output). For more information, please read the Radar user guide.

This method takes a class. This class is expected to be initialized with an Radar::ExceptionEvent, and must implement the `to_hash` method.

def reporter(klass)
  instance = klass.new
  yield instance if block_given?
  @reporters << instance
end
